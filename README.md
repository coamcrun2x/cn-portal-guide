# cn-portal-guide

本仓库用于归档和发布多个独立 HTML 页面，内容涵盖信息展示与导航用途。每个页面均为独立静态文件，不针对任何特定域名或具体网站。

## 项目简介

cn-portal-guide 是一个轻量级的 HTML 页面集合仓库，主要用于整理和发布各类信息门户风格的静态页面。所有页面均为独立文件，无需后端支持，可直接通过浏览器打开或部署至任意静态托管服务。

## 目录说明

```
cn-portal-guide/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面引用的静态资源（CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件代表一个独立页面，命名以功能或主题区分。
- `assets/`：公共资源文件，按类型分目录存放。

## 页面归档说明

- 每个页面均为完整 HTML 文档，包含样式与脚本（或引用公共资源）。
- 页面内容以信息展示为主，不含交互性后端逻辑。
- 归档目的为方便查阅与分发，不保证所有页面长期更新。
- 页面可能随时间增删或调整，请以仓库当前状态为准。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面通过 Pull Request 或直接推送管理。
- 欢迎提交 Issue 反馈问题或建议，但请注意本仓库不提供具体网站支持或替代方案。
- 如需引用或分发仓库内页面，请保留原始文件完整性，并注明来源。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/cn-portal-guide.git
   ```
2. 在浏览器中直接打开 `pages/` 目录下的 HTML 文件。
3. 或部署至任意静态服务器（如 GitHub Pages、Netlify 等）。
